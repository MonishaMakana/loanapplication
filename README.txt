Project: Multi-Step Loan Application

How to run the project
1. Open loan-app.xhtml in a modern browser
2. Complete the multi-step form by selecting a loan type and entering required details
3. Upload documents, draw the e-signature, accept terms, and submit

Tools used
- Browser (Chrome/Edge/Firefox)
- XHTML, CSS, JavaScript
- Browser localStorage for auto-save

Assumptions
- No server-side backend is required
- The project is designed for browser execution only
- Document upload and signature capture are client-side only
